## Descripción

Este repositorio recopila implementaciones y análisis de **técnicas avanzadas de clasificación de patrones**, abordando desafíos críticos como el **desbalance de clases**, la **reducción de dimensionalidad**, la **imputación de datos faltantes**, el **diagnóstico de underfitting/overfitting** y la **evaluación robusta mediante pruebas estadísticas**.

Desarrollado durante mi formación en el **Centro de Investigación en Computación (CIC) del IPN**, este repositorio sirve como referencia para futuros trabajos en clasificación, documentando buenas prácticas, experimentos reproducibles y análisis profundos del rendimiento de modelos.

---

## Objetivos

- **Analizar patologías en medidas de desempeño**: Identificar por qué métricas como accuracy pueden ser engañosas en datos desbalanceados.
- **Evaluar el efecto del desbalance de clases**: Implementar y comparar técnicas de balanceo (SMOTE, ADASYN, cost-sensitive learning).
- **Aplicar reducción de dimensionalidad**: Utilizar PCA, t-SNE, UMAP y selección de características para optimizar modelos.
- **Diagnosticar underfitting y overfitting**: Mediante curvas de aprendizaje, validación cruzada y análisis de sesgo-varianza.
- **Implementar imputación MICE**: Manejo robusto de datos faltantes mediante imputación múltiple encadenada.
- **Validar con pruebas estadísticas**: Aplicar pruebas como t-test, Wilcoxon, ANOVA y McNemar para comparar modelos.
